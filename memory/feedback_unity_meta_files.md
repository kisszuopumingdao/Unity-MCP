---
name: Do not manually create .meta files
description: Unity .meta files should never be created manually — they are auto-generated after assets-refresh or skill execution
type: feedback
---

Never manually create `.meta` files for Unity assets. They are auto-generated by Unity after `assets-refresh` tool/skill execution or any AssetDatabase.Refresh() call.

**Why:** Unity manages .meta files internally. Manually created ones may have incorrect format or conflicting GUIDs. The `assets-refresh` tool triggers proper auto-generation.

**How to apply:** When creating new `.cs` files or other assets, just create the file itself. Run `assets-refresh` (or let a skill that does AssetDatabase.Refresh() handle it) to generate the `.meta` file automatically.
